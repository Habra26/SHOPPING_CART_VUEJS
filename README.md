# Shopping Cart Vue.js

Un projet de panier d’achats réalisé en Vue.js dans le cadre de mon examen de framework client.

---

## Objectif

Ce projet avait pour objectif de :

- Concevoir une application web frontend de type boutique avec un panier.
- Permettre l’ajout et la suppression de produits dans le panier.
- Gérer et afficher l’état global de l’application (produits / panier).
- Utiliser les outils modernes de l’écosystème Vue.js (components, router, store…).

---

## Technologies utilisées

- Vue.js
- Vite
- JavaScript / HTML / CSS (fourni par le professeur)
- GitKraken & GitHub

---

## Fonctionnalités

- Liste des produits affichée dynamiquement.
- Ajout d’articles dans le panier.
- Mise à jour du nombre d’articles dans le panier.
- Calcul des totaux (quantité / prix / TVA / frais de livraison).
- Interaction réactive sans rechargement de page.
